  ---
  通用步骤

  1. 登录你们注册的 Redis 云平台
  2. 创建一个数据库实例（免费版即可）
  3. 在实例详情页找到连接信息，通常会直接给你一个 URL，格式类似：

  redis://default:<password>@<host>:<port>

  ---

  - 登录 → 点 "Create Database" → 选区域（建议选离你 Vercel 部署区域近的）
  - 创建后在 Details 页面直接能看到 UPSTASH_REDIS_REST_URL 和
  UPSTASH_REDIS_REST_TOKEN

  Upstash 还提供 HTTP 方式访问 Redis（@upstash/redis 包），在 Vercel serverless
  环境下比 TCP 连接更稳定，不存在连接池问题。

● Redis Cloud 不会直接给你一个完整的
  URL，你需要自己拼接。在数据库详情页找到这几个字段：

  在 Redis Cloud 控制台找连接信息

  1. 登录 redis.io/cloud → 进入你的 Subscription → 点击你创建的 Database
  2. 在数据库详情页（Configuration 或 General 区域）找到：
    - Public endpoint — 格式类似
  redis-12345.c1.us-east-1-2.ec2.redns.redis-cache.windows.net:12345（一个
  host:port）
    - Default user password — 点击眼睛图标复制密码

  拼接 REDIS_URL

  redis://default:<password>@<host>:<port>

  举例，如果你看到的是：
  - Endpoint:
  redis-18293.c258.us-east-1-4.ec2.redns.redis-cache.windows.net:18293
  - Password: AbCdEf123456

  那你的 URL 就是：
  redis://default:AbCdEf123456@redis-18293.c258.us-east-1-4.ec2.redns.redis-cach
  e.windows.net:18293

  ---
  拿到之后怎么配置

  把这个 URL 加到 Vercel 环境变量里：
  1. Vercel 项目 → Settings → Environment Variables
  2. 添加 REDIS_URL，值填上面拼好的 URL
  3. 勾选 Production / Preview / Development

  本地开发的话，在项目根目录创建 .env 文件加上同样的变量即可。
